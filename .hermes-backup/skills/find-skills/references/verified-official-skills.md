## red flags (SKIP these skills)

- Skills with <100 installs from unknown authors
- Skills claiming official orgs but with mismatched skill lists
- Repos with <100 stars
- Repos created very recently with no commit history
- Skills with suspicious names (typos of official orgs, etc.)
