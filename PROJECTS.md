# HermesAi — организация проектов

## Корневая структура

```
~/HermesAi/
├── AGENTS.md           ← личная инструкция (стиль, правила)
├── Vault/              ← Obsidian-подобные заметки, база знаний
├── Projects/           ← активные проекты
│   ├── project-1/
│   ├── project-2/
│   └── _template/      ← шаблон для новых проектов
├── Archives/           ← завершённые проекты
├── Logs/               ← логи, сессии, выводы
└── Scripts/              ← повторяемые скрипты
```

## Правила работы с проектами

1. Каждый проект — отдельная папка в Projects/
2. Имя папки: kebab-case, английские буквы, только a-z, 0-9, -
3. Внутри проекта — свой README.md с описанием задачи
4. Завершённые проекты перемещать в Archives/

## Структура одного проекта

```
projects/my-website/
├── README.md           ← цель, контекст, ссылки
├── notes.md              ← заметки, мысли
├── outputs/              ← результаты работы
├── assets/               ← файлы, картинки, данные
└── .env (опц.)           ← переменные, секреты проекта
```
