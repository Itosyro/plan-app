# Привет! 👋

Это твой **Obsidian-like vault** для Hermes AI.

## Как это работает

- Все заметки хранятся в `~/HermesAi/Vault/`
- Я умею читать, создавать и редактировать .md файлы
- Поддерживается Obsidian Flavored Markdown (wikilinks, callouts, embeds)

## Команды

| Команда | Что делает |
|---------|------------|
| `создай заметку [название]` | Новая заметка |
| `найди [запрос]` | Поиск по заметкам |
| `добавь в [заметка]` | Добавить контент |

## Структура

- [[Inbox]] — входящие заметки
- [[Projects]] — проекты
- [[Notes]] — общие заметки
- [[Archive]] — архив
