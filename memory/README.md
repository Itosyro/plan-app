# memory/

Папка для сырых потоков мыслей пользователя — для будущей оптимизации промптов через DSPy.

**Содержимое не коммитим в репозиторий** — это персональные данные. На production хранится либо в отдельной директории на Render volume, либо в S3-совместимом bucket.

В репо — только этот README, чтобы папка сохранялась в git.

## Формат (предполагаемый, уточняется в Phase 2)

```
memory/
  YYYY-MM/
    YYYY-MM-DD_HH-MM-SS_<inbox_id>.json
```

Где каждый JSON содержит:
- `inbox_id` — ID из БД
- `received_at` — UTC timestamp
- `kind` — text / voice
- `raw_text` — оригинальный текст (для текста)
- `transcript` — транскрипт (для голоса)
- `audio_path` — путь к аудиофайлу (для голоса)
- `pipeline_output` — результат AI-пайплайна
- `user_corrections` — если юзер потом правил руками — что именно
